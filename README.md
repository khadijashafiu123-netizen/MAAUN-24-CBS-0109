# Local Business Directory (Flask App)

## Description
This is a simple Flask web application that allows users to view and add local businesses. It also shows recently added businesses.

This project demonstrates:
- Object-Oriented Programming (OOP)
- Data Structures (Stack)
- Python datetime module
- Flask Web Framework
- GitHub Version Control

## Features
- View list of businesses
- Add new businesses
- See recently added businesses
- Timestamp for when a business was added

## How to Run the App

Step 1: Install Python

Step 2: Install Flask
pip install flask

Step 3: Run the application
python app.py

Step 4: Open browser and go to
http://127.0.0.1:5000/

## Project Structure
app.py – Flask application  
models.py – OOP classes  
templates/ – HTML pages
